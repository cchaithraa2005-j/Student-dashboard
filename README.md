
# 🎓 AIMS Student Dashboard (Power BI)

## 📊 Project Overview

The **AIMS Student Dashboard** is an interactive data visualization project built using **Power BI**. It provides insights into student demographics, academic distribution, and engagement metrics across different departments.

This dashboard helps in understanding student patterns and supports data-driven decision-making in educational institutions

## 📁 Dataset Summary

The dataset includes information about:

* Student ID
* Department (Computer Science, Commerce, Business Administration)
* Course (BCA, BBA, B.Com, BBA Aviation)
* Gender
* Age
* Year (1st, 2nd, 3rd)
* Learning methods
* Laptop availability
* Scholarship status
* Career goals
* Extracurricular participation
* Blood group

### ⚽ Extracurricular Participation
